## Examples

Graph description:

```haskell
A1 -- A2 @Edge1
A2 -- A3 @Edge2
A3 -- A4 @Edge3
A4 -- A5 @Edge4
A1 -- A4 @Edge5
A2 -- A4 @Edge6
A5 -- A6 @Edge7

```

### Launch 1: java Main --awt --list

### Launch 2: java Main --javafx --list

### Launch 3: java Main --javafx --matrix

### Launch 4: java Main --awt --matrix
